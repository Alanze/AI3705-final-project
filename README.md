### Prerequisites
- Linux or macOS
- Python 3
- CPU or NVIDIA GPU + CUDA CuDNN

### Getting started

Install Flask and other dependenies.

For pip users, please type the command `pip install flask`

- run `generated_code.py` to check the result in terminal and the web
